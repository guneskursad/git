branch
=============
- Branch is nothing but a pointer to a specific commit in GIT
- By default the branch used is "master"

- List all branches
# git branch

- Create a new branch
# git branch <NAME>

- Switch to an existing branch
# git checkout <NAME>

- Create a new branch and Switch to it
# git checkout -b <NAME>
# git switch -c <NAME>

- Delete a branch
# git branch -d <NAME>
# git branch --delete <NAME>
